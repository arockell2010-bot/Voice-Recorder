Voice Recorder App

This is a phone-installable web app. It records audio, shows a live meter,
saves recordings on the device, and lets you play, download, or delete takes.

How to use it on a phone:

1. Upload this folder to any HTTPS web host.
2. Open the site on the phone.
3. On iPhone: tap Share, then Add to Home Screen.
4. On Android Chrome: tap the menu, then Install app or Add to Home screen.
5. Open it from the home screen and allow microphone access.

Important:

- Microphone recording requires HTTPS, except while testing on localhost.
- Recordings are stored in the browser/app storage on that device.
- Download important recordings before clearing browser data or removing the app.
